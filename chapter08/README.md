# Chapter 08 Agent Architecture Examples

This chapter contains Java examples for Agent architecture design:

- `agent-concepts-demo`: models the minimum Agent capability set.
- `minimal-agent-loop-demo`: implements a hand-written Agent loop.
- `spring-boot-agent-runtime-demo`: shows a Spring Boot shaped Agent runtime.
- `langchain4j-agentic-workflow-demo`: shows a LangChain4j Agentic shaped workflow.

All examples default to DeepSeek and keep Ollama / ChatGPT configuration entry points.
