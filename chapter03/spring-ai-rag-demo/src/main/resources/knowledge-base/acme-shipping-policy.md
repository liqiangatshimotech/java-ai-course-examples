# ACME 实物赠品物流与签收规则

企业版活动赠品通常在客户完成合同签署后 7 个工作日内发出。物流单号由运营团队写入 CRM。

客户反馈未收到赠品时，客服需要核对合同状态、收件地址、联系人手机和物流单号。如果物流显示已签收，需要客户提供未收到说明并由运营团队向快递方发起核查。

赠品物流问题不属于退款问题，不能引导客户走退款流程。
