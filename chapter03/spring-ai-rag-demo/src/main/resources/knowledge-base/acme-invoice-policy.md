# ACME 发票开具与报销材料规则

电子发票支持企业抬头、个人抬头和专票申请。企业专票需要客户提供公司全称、纳税人识别号、注册地址、开户行和银行账号。

发票开具失败时，客服需要确认订单是否已支付成功、发票抬头是否完整、邮箱是否可用。如果支付状态未完成，不能承诺立即开票。

如果客户用于内部报销，客服可以提供订单明细、付款时间、发票状态和预计处理节点，但不能承诺财务审核一定通过。
