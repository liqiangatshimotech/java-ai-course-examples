# Acme 退款与重复扣费处理

企业版客户反馈重复扣费时，客服必须先核对订单号、支付流水号、扣费时间和客户租户 ID。确认重复扣费后，在工单中标记为 P1 财务问题，并转交财务支持团队。

重复扣费确认后，Acme 承诺在 1 个工作日内发起原路退款。退款发起后，银行卡通常 3-5 个工作日到账，企业网银可能需要 5-7 个工作日。

如果客户同时要求发票修正，需要先作废错误发票，再重新开具正确金额的发票。客服回复中必须附上退款单号和预计到账时间。
